# Tipos, variables, constantes, expresiones y operadores

En esta práctica trabajarás conceptos básicos de PHP relacionados con tipos de datos, variables, constantes, operadores y recepción de datos de formularios.

## Archivos de esta práctica

- 01-tipos.php
- 02-tipos-datos.php
- 03-variables.php
- 04-constantes.php
- 05-operaciones-aritmeticas.php
- 06-incremento-decremento.php
- 07-formulario-post.php

## Ejercicio 1: 01-tipos.php

1. Declara 3 variables de distinto tipo.
2. Crea una constante con tu nombre.
3. Haz una suma y muéstrala con `echo`.
4. Compara dos valores con `==` y con `===`.
5. Usa `var_dump()` para ver el tipo de cada variable.

## Ejercicio 2: 02-tipos-datos.php

1. Define una variable para cada tipo de dato: entero, flotante, cadena, booleano y array.
2. Muestra el tipo y el valor de cada variable usando `var_dump()`.

## Ejercicio 3: 03-variables.php

1. Declara una variable con un número entero y otra con una cadena de texto.
2. Cambia el valor de la variable entera.
3. Concatena un nuevo texto a la variable de texto.
4. Muestra ambas variables antes y después de la modificación.

## Ejercicio 4: 04-constantes.php

1. Define la constante `IVA` con el valor `0.21`.
2. Define la constante `TIENDA` con el nombre de tu tienda.
3. Calcula el precio final de un producto que cuesta 100 aplicando el IVA.
4. Muestra el nombre de la tienda y el precio final.

## Ejercicio 5: 05-operaciones-aritmeticas.php

1. Realiza suma, resta, multiplicación y división de dos números.
2. Muestra los resultados en pantalla.

## Ejercicio 6: 06-incremento-decremento.php

1. Declara una variable `$numero` con valor inicial `5`.
2. Realiza pre-incremento.
3. Realiza post-incremento.
4. Realiza pre-decremento.
5. Realiza post-decremento.
6. Muestra los resultados en cada paso.

## Ejercicio 7: 07-formulario-post.php

1. Crea un formulario con los campos `nombre`, `edad` y `suscrito`.
2. Recoge los datos en PHP usando `$_POST`.
3. Muestra un mensaje distinto según la información recibida.
4. Usa `var_dump()` para comprobar qué datos llegan realmente.
