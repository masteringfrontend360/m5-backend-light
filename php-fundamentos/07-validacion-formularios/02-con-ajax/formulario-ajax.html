Ejercicio 02 · Validación con AJAX y JSON

Objetivo: validar un formulario sin recargar la página.

El navegador envía los datos con fetch(), PHP responde en JSON y JavaScript pinta los errores o el éxito directamente en pantalla.

Trabajamos fetch(), FormData, trim(), strlen(), preg_match(), filter_var() y códigos HTTP como 422.

Formulario AJAX

Entre 2 y 50 caracteres. Solo letras y espacios.
Debe ser un email válido.

Qué está pasando aquí

  • El formulario no se envía de forma clásica (no hay recarga).
  • Se usa fetch() para enviar los datos.
  • PHP devuelve un JSON en vez de HTML.
  • JavaScript interpreta la respuesta y pinta errores o éxito.

Qué hemos mejorado (nivel PRO)

  • Controlamos errores cuando el servidor no devuelve JSON válido.
  • Evitamos que la aplicación se rompa si hay fallos en PHP.
  • Mostramos mensajes claros al usuario en cualquier escenario.
